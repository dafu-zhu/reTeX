# Test Data

Test PDFs for pipeline development and copyright filter research.

## Files

- `herman_pde_200.pdf` — First 200 pages of *Introduction to Partial Differential Equations* by Russell L. Herman (2015). Licensed under [CC BY-NC-SA](https://creativecommons.org/licenses/by-nc-sa/4.0/). Source: https://people.uncw.edu/hermanr/pde1/pdebook/PDE_Main.pdf

## Why this book?

- Freely licensed (CC BY-NC-SA) — safe to commit and test against
- Similar structure to typical math textbooks: chapters, sections, equations, figures, exercises
- Covers PDE topics (heat equation, wave equation, Fourier series, Sturm-Liouville) similar to books the pipeline was designed for
- 200 pages is enough to test all pipeline phases without being too large
